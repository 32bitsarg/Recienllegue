# Instrucciones de Sesión - Recién Llegué

## Enfoque de Trabajo
- **Prioridad Desktop**: Todos los cambios, optimizaciones y nuevas funcionalidades se asumen por defecto para la versión de escritorio.
- **Aesthetic Premium**: Mantener un diseño limpio, moderno y con micro-animaciones fluidas.
- **Optimización de Recursos**: Seguir usando `Next/Image`, `useMemo` y `memo` para mantener el alto rendimiento logrado.
- **Base de Datos Local**: Trabajar siempre primero en la base de datos local y pedir confirmación antes de cualquier deploy a producción.
- **Integridad de Datos**: Prohibido borrar datos sin autorización explícita.
